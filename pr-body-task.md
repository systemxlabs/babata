## 任务管理界面实现

### ✅ 已实现功能

#### 1. 根任务列表
- 展示所有根任务（支持分页）
- 每行显示：任务描述、Agent、状态、创建时间、子任务数
- 支持删除根任务（带二次确认弹窗）
- 支持根据状态筛选（全部/运行中/已完成/失败/已暂停/已取消）
- 支持按 Agent 筛选
- 支持按描述关键词搜索

#### 2. 任务树展示
- 可展开/折叠的树形结构
- 显示任务层级关系
- 显示每个任务的基本信息（描述、状态）
- 子任务懒加载，点击展开时获取

#### 3. 任务详情弹窗
- 点击任务可打开详情弹窗
- 标签页切换：
  - **任务目录**：显示任务目录下的所有文件，支持点击查看文件内容
  - **任务日志**：显示任务的执行日志，支持自动刷新
- 显示任务完整信息（ID、描述、Agent、状态、创建时间等）
- 支持任务控制操作（暂停/恢复/取消）

#### 4. 文件浏览器
- 树形文件目录展示
- 支持查看文件内容
- 代码高亮显示

### 🔌 新增 API 接口

| 接口 | 方法 | 描述 |
|------|------|------|
| /api/tasks/roots | GET | 获取根任务列表（支持分页和筛选） |
| /api/tasks/:id/children | GET | 获取任务的子任务列表 |
| /api/tasks/:id | DELETE | 删除任务（及其所有子任务） |
| /api/tasks/:id/control | POST | 控制任务状态（暂停/恢复/取消） |
| /api/tasks/:id/files | GET | 获取任务目录文件列表 |
| /api/tasks/:id/files/content | GET | 获取任务文件内容 |
| /api/tasks/:id/logs | GET | 获取任务日志 |
