# Babata Web UI 设计方案

## 1. 系统定位

Babata 是一个**多智能体多任务系统**，Web UI 需要清晰展示：
- 任务树结构（父子任务关系）
- 多智能体协作状态
- 多渠道接入（Telegram、WeChat 等）
- 技能管理

---

## 3. 配色方案

### 主色调
- **主色**: `#2563EB` (蓝色) - 代表智能、科技
- **成功**: `#10B981` (绿色) - 任务完成
- **警告**: `#F59E0B` (橙色) - 运行中、暂停
- **错误**: `#EF4444` (红色) - 失败、取消
- **中性色**: `#64748B` / `#94A3B8` - 文字、边框

### 背景色
- **深色模式**: `#0F172A` (slate-900)
- **卡片背景**: `#1E293B` (slate-800)
- **hover**: `#334155` (slate-700)

### 4.2 任务管理页面

**功能**：展示任务树，支持任务操作

**布局**：
```
┌─────────────────────────────────────────────────────────────┐
│  任务管理                                          [+ 新建] │
├─────────────────────────────────────────────────────────────┤
│  [全部 ▼] [状态: 全部 ▼] [智能体: 全部 ▼]      [🔍 搜索...] │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ▼ 📁 任务组 A (ID: xxx)                         [▶] [⏸] [✕]│
│    ├─ ● 子任务 A1        🟢 completed    planner   10:23   │
│    ├─ ● 子任务 A2        🟡 running      executor  10:15   │
│    └─ ▶ 子任务 A3        ⚪ pending      reviewer  --:--   │
│                                                             │
│  ▶ 📁 任务组 B (ID: xxx)                         [▶] [⏸] [✕]│
│                                                             │
│  ▼ 📁 任务组 C (ID: xxx)                         [▶] [⏸] [✕]│
│    ├─ ● 子任务 C1        🔴 failed       executor  09:45   │
│    └─ ● 子任务 C2        🟢 completed    planner   09:40   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**任务状态颜色**：
- 🟢 completed - 绿色
- 🟡 running - 黄色/蓝色动画
- ⚪ pending - 灰色
- 🔴 failed - 红色
- ⏸️ paused - 橙色
- ⚫ canceled - 灰色删除线

**交互**：
- 点击任务展开详情面板（右侧滑出）
- 右键菜单：暂停/恢复/取消/删除
- 拖拽调整任务顺序

## 5. 技术选型建议

### 前端框架
- **React 19** + **TypeScript** (已使用)
- **状态管理**: Zustand 或 Jotai (轻量级)
- **路由**: React Router v7

### UI 组件库
- **推荐**: 
  - **Shadcn/ui** - 现代化、可定制
  - 或 **Ant Design** - 功能完整、成熟
  - 或 **Mantine** - 开发者体验好

### 图表库
- **Recharts** - React 原生，易用
- 或 **ECharts** - 功能强大

### 样式方案
- **Tailwind CSS** - 原子化 CSS，开发效率高
- 已集成到项目中

### 图标
- **Lucide React** - 现代、简洁

## 6. 响应式设计

### 断点设计
- **Desktop**: ≥1280px - 完整布局
- **Laptop**: ≥1024px - 侧边栏可收起
- **Tablet**: ≥768px - 侧边栏变底部导航
- **Mobile**: <768px - 单列布局，抽屉导航

## 8. 交互细节

### 动画效果
- 页面切换：淡入淡出 200ms
- 任务状态变化：脉冲动画
- 侧边栏展开：滑动 300ms ease-out
- 卡片 hover：轻微上浮 + 阴影加深

### 实时更新
- WebSocket 连接后端
- 任务状态实时推送
- 新消息通知

### 快捷键
- `Ctrl+K` - 全局搜索
- `Ctrl+N` - 新建任务
- `R` - 刷新数据
