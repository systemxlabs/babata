## Summary
Remove the redundant "role" field from the message store table schema.

## Changes
- Remove "role" column from messages table schema
- Remove role parameter from INSERT queries
- Remove role field from SELECT queries
- Remove Role enum from message.rs
- Remove Message::role() method
- Remove role validation check when scanning messages

## Testing
- ✅ All 115 tests pass
- ✅ cargo clippy clean
- ✅ cargo fmt applied

## Breaking Change
This change modifies the database schema. Existing databases with the role column will need to be recreated or migrated.
